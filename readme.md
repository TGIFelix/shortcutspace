This app is currently in development.
You can contribute by adding shortcuts using the template below.

```markdown
| group |
| --- | --- |
| CMD | description |
| CMD | description |
| CMD | description |

```

for example:

```markdown
| general |
| --- | --- |
| ctrl+x, ctrl+f | open file |
| ctrl+x, ctrl+s | save |
| ctrl+x, ctrl+w | save as |

```
